You can use lumberjacking for either lumbering wood or to farm crops.

To lumber wood you need to double-click any type of axe then select a tree to lumber from.

Higher lumberjacking skill and specialized crafter class will generate more logs, lumberjacking with Xarafax’s axe will double the amount of logs you get.

Having atleast 90 lumberjacking and being specialized crafter class will make you handle the axe so good that it won’t break again.

To farm crops you will need to double-click the crop you want to farm.
