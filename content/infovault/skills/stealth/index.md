The stealth skill allows you to move while being hidden.

To use stealth you will first need to be hidden and have at least 80 hiding.

If you run or use any skill/item while in stealth you will break it and reveal yourself.

The higher stealth skill and specialized thief class you are the longer you can move in stealth.
