Cooking is used by double-clicking the ingredient to what you want to make near the correct type of fire.

The food value shows how much the food type will fill you up when eaten.

The cooking method determines wich fire type you will need.

## Cooking Method

| Method | Fire type                            |
|--------|--------------------------------------|
| 0      | None                                 |
| 1      | Requires a fire                      |
| 2      | Requires an oven                     |
| 3      | Can be cooked in both oven and fires |
